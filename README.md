# ✈️ AI-Travel-Guide (HTML Single Page)

> **打造您的专属数字旅行手账：杂志级审美、全交互、单文件自动保存。**

## 📖 项目简介 | Introduction

这是一个精心设计的 AI Prompt（提示词）工程。旨在让 ChatGPT (GPT-4) 或 Claude (3.5 Sonnet) 化身为一位**资深全栈工程师兼旅行主编**。

通过此提示词，您可以快速生成一个**单文件 HTML 旅行指南**。它不仅仅是一张静态的网页，而是一个具备**本地存储、拖拽排序、所见即所得导出**功能的现代化 Web App。

## ✨ 核心亮点 | Features

*   **🎨 杂志级排版**：基于 Tailwind CSS 的玻璃拟态（Glassmorphism）与衬线体排版，视觉效果惊艳。
*   **💾 自动保存**：基于 `localStorage` 的实时保存机制，刷新页面不丢失数据，修改内容（文本/图片/机位）即刻同步。
*   **🖱️ 深度交互**：
    *   支持所有内容的**拖拽排序** (SortableJS)。
    *   支持时间卡片**独立换色**。
    *   点击即可编辑所有文字 (ContentEditable)。
*   **📤 完美导出**：内置 `html2pdf` 与 `html2canvas`，支持一键生成高清 **PDF** 或 **长图**，自动隐藏 UI 控件。
*   **🚀 零依赖部署**：所有逻辑封装在单个 `.html` 文件中，双击即可运行，无需服务器，方便分享。

## 🛠 使用方法 | Usage

1.  **复制提示词**：复制下方 `Prompt.md` 中的所有内容。
2.  **发送给 AI**：推荐使用 **gemini-3-pro-preview** (效果最佳) 或 **豆包**。
3.  **交互生成**：
    *   AI 会进入“深度信息收集”阶段，按步骤询问您的旅行计划。
    *   回答完毕后，AI 将生成完整的 HTML 代码。
4.  **保存运行**：
    *   新建文本文件，粘贴代码，保存为 `.html` (例如 `Tokyo_Trip.html`)。
    *   浏览器打开即可使用。
